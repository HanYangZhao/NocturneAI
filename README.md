# NocturneAI


# Installation

## Install venv
```python -m venv venv```
### In cmd.exe
```venv\Scripts\activate.bat```
### In PowerShell
```venv\Scripts\Activate.ps1```
### In Linux/MacOs
```source myvenv/bin/activate```

## Install packages
```pip install -r requirements.txt```


## Create a .env file
```
OPEN_API_KEY=YOUR_KEY
ELEVEN_LABS_API_KEY=YOUR_KEY
```

# Run the program
```python gui.py```
